1.3  Biểu đồ Trường hợp Sử dụng (Use Case Diagram)

Lưu ý: Hệ thống không có chức năng đăng nhập cho nhân viên thông thường. Chỉ Super Admin, Admin và Manager mới có thể đăng nhập vào ứng dụng web. Nhân viên được hệ thống nhận diện tự động qua thiết bị Raspberry Pi 4 mà không cần thao tác gì.

Chú thích (Legend)

Super Admin — Toàn quyền hệ thống (kế thừa Admin qua «generalize»)
Admin — Quản lý toàn bộ nghiệp vụ (CRUD nhân viên, phòng ban, ca, thiết bị)
Manager — Xem & chỉnh sửa điểm danh phạm vi phòng ban
Raspberry Pi 4 — Tác nhân thiết bị, giao tiếp qua Device API (token)
──▶ «include» : UC này bắt buộc gọi UC kia
──▶ «extend» : UC này mở rộng UC kia (tùy điều kiện)
«generalize» : Super Admin kế thừa mọi quyền của Admin
Ellipse nét đứt = Use Case được gọi gián tiếp qua include/extend
