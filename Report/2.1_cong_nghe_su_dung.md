## 2.1. Công nghệ sử dụng

**Frameweb:** Laravel 13 (PHP 8.3+) — framework MVC phía server, cung cấp REST API cho thiết bị Raspberry Pi và giao diện web quản lý; xác thực phân quyền qua Laravel Sanctum và Laravel Breeze.

**Frontend:** Blade Template Engine (Laravel) kết hợp TailwindCSS 3 để dựng giao diện; Alpine.js 3 xử lý tương tác động phía client; Vite làm công cụ build và hot-reload; Chart.js hiển thị biểu đồ thống kê; DataTables.js quản lý bảng dữ liệu phân trang.

**Dữ liệu:** MySQL 8.0 lưu trữ toàn bộ dữ liệu quan hệ (nhân viên, điểm danh, phòng ban, ca làm việc); SQLite làm bộ đệm cục bộ trên thiết bị Pi 4 khi hoạt động ngoại tuyến.

**Học máy:** Thư viện `face_recognition` 1.3.0 (dựa trên dlib và mạng nơ-ron ResNet-34) để trích xuất vector đặc trưng 128 chiều và so khớp khuôn mặt; OpenCV 4.9.0 để thu nhận khung hình từ camera và tiền xử lý ảnh.

**Các thư viện khác:** `numpy` 1.26.4 (tính toán vector), `requests` 2.31.0 (giao tiếp HTTP từ Pi lên server), `python-dotenv` 1.0.1 (quản lý biến môi trường), PhpSpreadsheet 2.0 (xuất Excel), DomPDF 3.1 (xuất PDF).
